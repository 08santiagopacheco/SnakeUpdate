🐍 Snake Evolution: Level-Based Challenge
¡Bienvenido a Snake Evolution! Este no es el clásico juego de la serpiente de los años 90. Es una versión modernizada desarrollada en Python, diseñada con una arquitectura de código limpia y un sistema de progresión por niveles que desafiará tus reflejos.

✨ ¿Qué hace a este Snake diferente?
A diferencia de las versiones básicas, este proyecto incluye:

📈 Sistema de Niveles: Supera objetivos de puntuación para avanzar al siguiente nivel. ¡La dificultad aumenta dinámicamente!

⚡ Velocidad Variable: La serpiente acelera conforme avanzas, exigiendo mayor precisión.

🎨 Interfaz Visual Mejorada: Colores vibrantes y una gestión de colisiones más fluida.

💾 Persistencia de Datos: Sistema de puntuación máxima para que siempre tengas un récord que batir.

🛠️ Tecnologías utilizadas
Lenguaje: Python 3.x

Librería Gráfica: Pygame (o Tkinter, cámbialo según tu caso)

Lógica de Juego: Programación Orientada a Objetos (POO) para facilitar la escalabilidad.

🚀 Instalación y Ejecución
Si quieres probar el juego en tu máquina local, sigue estos pasos:

Clona el repositorio:

Bash

git clone https://github.com/TU_USUARIO/snake-evolution.git
Instala las dependencias:

Bash

pip install pygame
Ejecuta el juego:

Bash

python main.py
🎮 Controles
Flechas de dirección / WASD: Mover la serpiente.

Espacio: Pausar el juego.

R: Reiniciar nivel.
